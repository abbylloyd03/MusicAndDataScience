# The Foundational Guide to Data Science & AI Applications in Music

This project provides an interactive Streamlit web application featuring introductory tutorials on open-source analytics and AI tools for music generation and analysis. These tutorials are designed to help music students and enthusiasts gain fundamental skills needed to understand current AI applications in music.

## üéµ Features

### Interactive Tutorials
The app includes comprehensive tutorials covering:
- **Hello Melody** - Introduction to creating melodies with music21
- **Building & Analyzing Chords** - Understanding chord construction and analysis
- **MusicXML Visualization** - Interactive app for visualizing music notation
- **Chordify & Key Inference** - Automatic chord detection and key identification
- **Loading & Visualizing Audio** - Working with audio files
- **Beat Detection & Tempo Analysis** - Rhythm and tempo analysis techniques
- **Pitch & Frequency Analysis** - Understanding pitch and frequency in audio
- **Audio Effects & Exporting** - Applying effects and exporting processed audio
- **Onset Detection ML** - Machine learning application for musical onset detection

### Technology Stack
- **Streamlit** - Interactive web application framework
- **music21** - Music analysis and generation library
- **librosa** - Audio and music analysis
- **scikit-learn** - Machine learning for onset detection
- **pandas, numpy** - Data manipulation and analysis
- **matplotlib, seaborn** - Data visualization
